# 🚀 CareerBridge – Internship & Placement Portal

CareerBridge is a full-stack web application designed to help students prepare for placements and internships. It provides resume analysis, internship discovery, learning roadmaps, and placement preparation resources through a modern and responsive interface.

---

## 🌐 Live Demo

* **Frontend:** https://career-bridge-git-main-vyshnavi6.vercel.app
* **Backend API:** https://careerbridge-backend-65et.onrender.com

---

## ✨ Features

* 🔐 User Authentication (Login & Signup)
* 📄 Resume Upload & ATS Resume Analysis
* 📊 Resume Score Dashboard
* 📥 Resume Report Download
* 💼 Internship Portal
* 🏢 Company Showcase
* 📚 Learning Roadmaps
* 🎯 Placement Preparation
* 📱 Fully Responsive UI
* ☁️ Cloud Deployment using Vercel & Render

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/vyshnavi7-coder/CareerBridge.git
```

Go to project directory

```bash
cd CareerBridge
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `backend` folder.

```
PORT=5000

MONGO_URI=YOUR_MONGODB_CONNECTION_STRING

JWT_SECRET=YOUR_SECRET_KEY
```
